# com.animiq.miq

Unity SDK for exporting and loading `.miq` avatar containers.

## Quickstart (5 minutes)

1. Open a supported Unity LTS project (Built-in Render Pipeline).
2. Add this package through UPM.
3. Select an avatar root object in Scene.
4. Export via `Tools/Animiq/MIQ/Export Selected AvatarRoot`.
5. Load at runtime:

```csharp
using Animiq.Miq.Runtime;

if (MiqRuntimeLoader.TryLoad(path, out var payload, out var diagnostics))
{
    UnityEngine.Debug.Log($"Loaded avatarId={payload.Manifest.avatarId}");
}
else
{
    UnityEngine.Debug.LogError(
        $"Load failed: code={diagnostics.ErrorCode}, stage={diagnostics.ParserStage}, msg={diagnostics.ErrorMessage}");
}
```

## Supported matrix (v1.0.0)

- Unity (official gate-backed lines):
  - `2021.3.18f1` (`2021-lts`)
  - `2022.3.62f1` (`2022-lts`)
  - `2023.2.20f1` (`2023-lts`)
- Package minimum floor: `2021.3.18f1`
- Render pipeline: Built-in RP only
- Shader allowlist: `Standard`, `MToon`, `lilToon`, `Poiyomi`
- File format support:
  - export: `v4`, `v5` (compression enabled writes `v5`)
  - load: `v1..v5`

Details: [compatibility.md](../../../docs/public/compatibility.md)

## Runtime API contract

- `MiqRuntimeLoader.Load(path)`
  - throw-on-failure path.
- `MiqRuntimeLoader.TryLoad(path, out payload, out diagnostics)`
  - non-throwing path.
- `MiqRuntimeLoader.TryLoad(path, out payload, out diagnostics, options)`
  - policy-based path using `MiqLoadOptions`.

Default load options:

- `StrictValidation = false`
- `UnknownSectionPolicy = Warn`
- `ShaderPolicy = WarnFallback`

Diagnostics contract:

- `ErrorCode`
- `ErrorMessage`
- `ParserStage`
- `IsPartial`
- `Warnings`
- `WarningCodes`

Error catalog: [error-codes.md](../../../docs/public/error-codes.md)

## Troubleshooting

- `MagicMismatch`: file is not MIQ or bytes are corrupted.
- `UnsupportedVersion`: generated by an unsupported format version.
- `MissingRequiredManifestKeys`: required manifest keys are missing.
- `ParityContractViolation`: strict shader/typed material policy mismatch (`ShaderPolicy = Fail`).
- Unknown sections:
  - `Warn` default: continue with warning code `MIQ_UNKNOWN_SECTION`.
  - `Fail`: stop at first unknown section.
